### aspira.go

Um pequeno slackbot

* https://slack.com/services/new/incoming-webhook
* https://slack.com/services/new/outgoing-webhook

### Desenvolvimento

* go get github.com/go-martini/martini
* go run main.go

Obs: Edite o arquivo aspira/util/util.go e coloque seus dados de servidor
